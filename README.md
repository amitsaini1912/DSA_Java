## 📆 Daily Log Format

Each day is recorded in its own folder (e.g., `Day01`, `Day02`, ...).  
Inside each folder, you'll find:

- ✅ Problem Statement (as a comment or `README.md`)
- 🧠 Solution/Code
- 📝 Approach and Time/Space Complexity (in comments)
